# 1. Barcode Standards

Support:

* Code 128 (default)
* QR Code
* Code 39
* EAN-13
* UPC-A

Default:

Products → Code 128

Assets → QR Code (configurable)

Administrator can change the default type.

# 2. Automatic Barcode Generation

Instead of requiring manual barcode entry, support automatic generation.

Example:

Products

PRD-000001

PRD-000002

PRD-000003

Assets

AST-000001

AST-000002

Administrator can configure:

* prefix
* starting number
* padding length
* generation format

Manual barcode entry must still be supported.

Never generate duplicate barcodes.
